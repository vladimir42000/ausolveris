# User Guide

## Status

This is the initial user-guide scaffold.

## Intended audience

This guide is for users who want to model loudspeaker-related problems using the platform, especially through YAML project definitions and reproducible study execution.

## Planned guide structure

- installation,
- project layout,
- first simulation,
- YAML basics,
- geometry and mesh input,
- outputs and exported data,
- interpretation notes,
- troubleshooting.

## Initial usage promise

The platform should eventually allow a user to describe a modeling problem in YAML, run it through a stable interface, and obtain traceable outputs without hand-editing internal code.
