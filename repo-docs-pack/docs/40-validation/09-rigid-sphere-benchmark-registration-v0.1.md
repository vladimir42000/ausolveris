# Analytical BEM Benchmark Registration (v0.1)

## Status
**Defined – BEN-004**

## Purpose
BEN-004 registers the `ben004_rigid_sphere_scattering_registered` benchmark descriptor to ensure that analytical validation metadata, source citations, and required reference fields are firmly defined in the architecture *before* any BEM physics are implemented.

## Hard Exclusions
- This task strictly forbids execution. 
- No BEM matrix assembly, no Green-function integration, and no SPL evaluation.
- The descriptor is explicitly locked to `registered_not_executed`.
