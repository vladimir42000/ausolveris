# ADR-0004 Append-over-overwrite document policy

- Doc ID: ADR-0004
- Version: v0.1
- Status: Accepted
- Authoring role: Documentation Steward
- Decision owner: Director
- Physics review required: No
- Audit required before acceptance: No
- Last updated: 2026-04-15

## Decision

Stable project documents must not be silently rewritten. Material meaning changes require dated updates, addenda, or superseding records.
