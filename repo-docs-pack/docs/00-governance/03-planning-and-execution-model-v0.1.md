## Core sequence

1. Governance setup.
2. Discovery and feasibility.
3. Architecture exploration.
4. Decision checkpoint.
5. Contract and specification drafting.
6. Task decomposition.
7. Coding.
8. Audit and merge.

## Preparation estimate

A realistic lean preparation period before coding is about 5 to 10 focused man-days, potentially 7 to 12 if discovery and feasibility need deeper review.

## Rule

No developer work begins before the first direction is accepted and first task packets exist.
