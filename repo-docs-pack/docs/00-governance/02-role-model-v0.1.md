# Role Model

- Doc ID: GOV-02
- Version: v0.1
- Status: Draft
- Authoring role: Director support / Documentation Steward
- Decision owner: Director
- Physics review required: No
- Audit required before acceptance: Yes
- Last updated: 2026-04-15

## Roles

### Director
Owns mission, priorities, scope acceptance, and final go/no-go decisions.

### Product Physicist
Owns physical realism, assumptions, benchmark relevance, and modeling legitimacy.

### Spec Lead
Owns architecture, interfaces, subsystem boundaries, and task granularity.

### Developer
Owns local implementation within approved contracts and task packets.

### Auditor
Owns acceptance checks against task, tests, docs, and merge gate.

### Documentation Steward
Owns traceability, document hygiene, version visibility, and cross-link consistency.

## Rule

One person or AI may temporarily draft across several roles during discovery, but ownership and review responsibilities must remain explicit.
