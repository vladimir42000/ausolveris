# AuSolveris

## Capability Disclaimer
AuSolveris currently provides geometry/readiness scaffolding and bounded sanity hooks.
- It is not a validated BEM solver.
- It does not yet implement a validated LEM driver/enclosure solver.
- It does not compute SPL curves.
- It does not compute impedance curves.
- It does not compute validated frequency response.
- It does not optimize loudspeaker designs.
- Current acoustic outputs are limited to explicitly marked sanity/stub formulations.
